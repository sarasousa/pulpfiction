We are a group of students of Electrical and Computers Engineering from FEUP. This website is our Multimedia Systems' project. We chose to make a website about Pulp Fiction, because it is one of our favorite movies and we have great respect and admiration for sir Quentin Tarantino. Our biggest professional interests are developing web applications and games.

We decided to make this application a static web page. The reason behind this is the possibility of storing the application without the need for a host or server. As such, it can be run from any location, provided you have internet access to load the aesthethic additions.
